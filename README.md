# MEVA (Materials Engineering Visualization and Analysis)

## 📖 프로젝트 소개

**MEVA**는 재료공학 학생들을 위한 재료실험 데이터 시각화 분석 프로그램입니다.
인장시험의 핵심 물성값을 이론 모델 기반으로 시뮬레이션하여 학습 효과를 높이는 것을 목표로 합니다.

## 🎯 개발 배경

- 재료 공학 실험 데이터 분석의 비효율성 해소
- 인장-항복-파단 과정의 이론적 거동 시뮬레이션 및 시각화
- 고가이고 복잡한 범용 소프트웨어 대신 학부생 수준에 최적화된 경량화 솔루션 제공
- Java Swing 기반의 간단하고 직관적인 GUI로 접근성 향상

## ✨ 주요 기능

### 1. 인장 시험 이론 시뮬레이션
- 사용자 입력 실험값 기반 응력-변형률 곡선 계산
- 재료의 탄성, 항복, 소성 경화, 파단 과정 실시간 시각화
- 이론적 거동 분석 및 결과 제공

### 2. 물성값 계산 및 출력
- 영률(Young's Modulus), 항복 강도, 인장 강도 계산
- 그래프 상 주요 물성값 표시
- 수치 결과 출력 및 해석 도움

### 3. 데이터 관리
- CSV/Excel 파일 Import/Export 기능
- 입력 물성치 및 시뮬레이션 결과 데이터 외부 활용
- PNG, JPG, SVG 형식 그래프 이미지 저장
- PDF 보고서 생성
