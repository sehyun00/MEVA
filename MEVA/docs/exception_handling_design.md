# 예외 처리 설계서

## 개요

시스템 오류를 안정적으로 처리하여 프로그램 중단을 방지하고 사용자에게
명확한 메시지를 제공하기 위한 설계 문서.

## 예외 처리 원칙

-   프로그램 중단 방지\
-   사용자에게 명확한 메시지 제공\
-   오류 로그 기록

## 처리 대상 예외

### 데이터 검증 실패

-   null 값\
-   범위 초과\
-   음수 입력

### 파일 처리 예외

-   존재하지 않는 파일\
-   지원하지 않는 형식\
-   파일 읽기/쓰기 실패

### 시스템 예외

-   OutOfMemoryError\
-   IllegalStateException

## 예외 처리 방식

### 사용자 메시지 예

-   "입력값이 범위를 벗어났습니다." \### 내부 로그 예
-   `ERROR [FileLoader] CSV parsing failed at line 34`
