# 검증 로직 설계서

## 1. 개요
사용자 입력 데이터와 계산 결과가 정확하고 일관적인지 검증하는 로직 설계.

## 2. 검증 대상
- 입력 데이터: 길이, 하중, 단면적
- 계산 결과: 인장강도, 변형률, 허용 하중 대비 안전율
- 파일 입출력: 데이터 파일 형식 및 값 범위

## 3. 검증 로직

### 3.1 입력값 유효성 검증
1. 데이터 범위 체크
   - 하중: 0 < F ≤ 10000 N
   - 길이: 0 < L ≤ 500 mm
   - 단면적: 0 < A ≤ 100 cm²
2. 데이터 형식 체크
   - 숫자 외 문자는 오류
   - 필수 입력값 누락 시 오류
3. 논리적 일관성 체크
   - 길이, 하중, 단면적 간 계산 가능 여부 확인

### 3.2 계산 결과 검증
- 인장강도 = 하중 / 단면적
- 변형률 = (최종 길이 - 초기 길이) / 초기 길이
- 이론적 한계값(예: SS400)과 비교
  - 범위 초과 → 경고/오류 메시지 출력

### 3.3 오류 처리
- 잘못된 입력 → 사용자에게 메시지 출력 및 해결 가이드 제공
- 파일 읽기 실패 → 로그 기록 후 프로그램 종료
- 계산 결과 이상치 → 로그 및 경고 메시지 출력

### 3.4 자동 검사 절차
- 반복적 테스트를 위해 검증 스크립트 설계
- 단위/통합 테스트 자동 실행 가능 구조
