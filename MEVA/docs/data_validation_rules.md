# 데이터 검증 규칙 명세서

## 개요

MEVA 시스템에서 입력되는 데이터의 신뢰성을 보장하기 위해 필요한 검증
규칙을 정의한다.

## 검증 대상

-   재료(Material) 속성 값\
-   실험 데이터 포인트\
-   파일 확장자\
-   설정(Config) 값\
-   사용자 입력 데이터

## 재료 속성 검증

### Young's Modulus

-   허용 범위: **1.0 \~ 1000.0 GPa**
-   음수 또는 0 불가

### Yield Strength

-   0보다 커야 함

### Tensile Strength

-   0보다 커야 함\
-   Yield Strength보다 낮으면 오류

## 실험 데이터 포인트 검증

-   최대 허용 개수: **10,000개**

## 파일 형식 검증

### 이미지 형식

-   허용: PNG, JPG, SVG\
    \### 데이터 파일 형식
-   허용: CSV, XLSX

## Config 설정 검증

### 창 크기

-   width ≥ 300\
-   height ≥ 300 \### 계산 정밀도
-   1\~10

## 사용자 입력 검증

-   공백 입력 금지\
-   숫자 입력 시 문자 포함 여부 검사\
-   파일 존재 여부 확인
